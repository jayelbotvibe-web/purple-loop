Architecture

The engine is built on five pluggable Go interfaces:

InterfaceRoleImplementations
FeedLoads prioritized techniquesStaticFeed, ArbiterFeed, EmulationFeed
ExecutorRuns atomics on targetsDockerExecutor, SSHExecutor, DryExecutor
CollectorQueries Wazuh for telemetryWazuhCollector (archives.json)
EvaluatorMatches Sigma rules against eventsRuleMatcherEvaluator
ReporterProduces coverage outputJSONReporter, HTMLReporter, NavigatorLayerReporter

Verdict Semantics

VerdictMeaning
DETECTEDSigma rule condition matched an event
PARTIALPrimary selection matched, secondary didn't
MISSEDEvents collected, no rule matched — real gap
NO_TELEMETRYZero events in window — collection problem
INCONCLUSIVEPipeline canary failed — run not trusted
